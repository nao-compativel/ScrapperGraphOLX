## FLUXO DE USO



Instale os pacotes que **não são da biblioteca padrão** do Python.

```
pip install pandas beautifulsoup4
```

Se estiver usando Python 3 explicitamente

```
pip3 install pandas beautifulsoup4
```

Se quiser garantir no ambiente correto

```
python -m pip install pandas beautifulsoup4
```



### Siga os passos:

1. Execute:  python #scrapPaginas_v4.py

2. Uma janela abrirá para você selecionar os arquivos .txt/.html das páginas OLX

3. Os dados novos serão mesclados ao dados_olx.js existente (sem apagar o histórico)

4. Abra AnalisadorOLX.html no navegador
